An online MUD game developed with a React frontend and Python/Django backend. It utilizes WebSockets for in-game communication via Pusher service.

The client code can be found [here](https://github.com/tgreenhalgh/LambdaMUD-Client). 
The server code can be found [here](https://github.com/tgreenhalgh/LambdaMUD-Project).

The deployed app can be found [here](https://mudclient.now.sh).
